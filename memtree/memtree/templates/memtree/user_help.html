MemTree

A simple application for text notes.

The main feature is the possibility of infinite nesting of elements.


Description of management:

  • Green highlighting (first press on element)

    Only one element can be marked in green.

    • To add new root (first) element:

      Press the button "Add" without green highlighting other elements.

    • Add new nested element (child):

      Press the button "Add" when the parent element is highlighted in green.

    • Edit text of element:

      Press the button "Edit" when the element is highlighted in green.

  • Yellow highlighting (second press on element)

    Multiple elements can be marked in yellow at the same time.

    • Delete selected (yellow) elements:

      Mark the elements in yellow and press "Del".

    • Move selected (yellow) elements to another element (parent):

      Mark elements in yellow.

      Then mark parent element in green.

      And press the button "Move".

      If the parent element is not selected, the yellow elements will become the top of the "tree".
